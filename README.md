# QR Code Scanner

*QR Code Scanner - a simple, fast and useful progressive web application*

Forked from https://github.com/gokulkrishh/qrcodescan.in

## Installation

1. Clone this repo

  ```bash
  git clone https://github.com/tractr/qrcodescan
  ```

2. Installation

  ```bash
  npm install
  ```

3. Run

  ```bash
  npm run start
  ```

4. Build

  ```bash
  npm run build
  ```
